Learn GRB Analysis in 3 days
=======================================

Conservatively speaking, Gamma-ray burst (GRB) data analysis can be learned in 3 days, even you can finish the learning within 3 hours if you have experience in programming or data analysis, just follow this guide. 

This guide is practical, its purpose is to learn enough technique to start your work, also it provides some links for deepening your knowledge. Hope it can acts as the entry of your new GRB data world.
